The following Python listings are reference implementations used to generate the main-text Figures 2–4. Scripts use the diffusion (D) and consumption rate constant (k) parameters reported in Table 1 (main text) / Table S2 (Supplementary Appendix). Time-to-threshold panels (Figure 2) are computed under a step boundary exposure (optimistic upper bound) by explicit finite-difference simulation of the two-sided slab diffusion–reaction model, with analytic steady-state reachability checks (Css_center ≥ M) used to classify “not reachable” cases.
